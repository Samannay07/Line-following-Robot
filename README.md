# Line-following-Robot
Arduino-based Line Following Robot using IR sensors and L298N Motor Driver.

## Overview

This project implements an autonomous Line Following Robot using Arduino and IR sensors.

## Components Used

- Arduino Nano
- IR Sensors
- L298N Motor Driver
- DC Motors
- Battery Pack
